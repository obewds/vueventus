Blazing Fast Prototyping

VueVentus components are bundled with Tailwind CSS classes and Vue.js component props for logical groupings of visual/aesthetic classes.

🧰

Highly Customizable

Each VueVentus component can be customized from a single root config file, giving a bird's eye view of the visual design aspects of your app.

⏱️

Production Focused

You are in control of your atomic class configurations, and can prune CSS payloads down to DRY goodness on both macro and micro levels.
